# Платформа Базальта
Предназначена для организации единого информационного пространства небольшого производственного предприятия.
## Контроль доступа
### Вход
Аутентификация пользователй осуществляется по логину и паролю
### Авторизация
В системе возможно создание и администрирование неограниченного количества пользовательских групп.
Авторизация пользователей осуществляется на основе прав, назначенных как непосредственно пользователю, так и группам, в которые он входит.
Уровни доступа:
* Нет доступа
* Просмотр
* Редактирование (удаление)
## Справочники
В типовой конфигурации система поддерживает ведение следующих справочников:
* Справочник материалов
* Справочник сортаментов и их типоразмеров
* Справочник стандартных изделий
* Справочник разработчиков
* Справочник поставщиков

## Работа с составом
В системе реализована возможность управления составом изделий по разделам конструкторской спецификации:
* Документация
* Комплексы
* Сборочная единица
* Детали
* Стандартное изделие
* Прочие изделия
* Материалы
* Комплекты
### Хранение информации об извещениях об изменениях

## Управление файловым архивом
Система позволяет управлять файловым архивом электронных документов, а также информацией о бумажных документах, если таковые еще используются предприятием.
Документация может хранится в привязке к элементам конструкции изделий, отслеживается версионность документов.
